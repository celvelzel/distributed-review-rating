Technical Highlights

  • DeBERTa Integration: Leveraged advanced transformer models for deep textual feature extraction.
  • LightGCN Graph Embeddings: Captured user-item collaborative signals using state-of-the-art graph neural networks.
  • K-Fold Target Encoding: Prevented data leakage while effectively handling high-cardinality categorical features.
  • Multi-level Stacking: Combined diverse model outputs to enhance overall prediction robustness.
  • Multimodal Architecture: Effectively unified text, graph, and statistical features into a cohesive pipeline.
  • PySpark Preprocessing: Handled large-scale data transformation in a pseudo-distributed environment.

Challenges & Weaknesses

  • Hardware Constraints: Relied on a single local GPU, restricting the scale of hyperparameter tuning.
  • Time Limitations: Strict project deadlines limited the depth of exploration for alternative deep learning architectures.
  • Cold Start Problem: Difficulty in predicting accurately for completely new users or products with minimal history.
  • Feature Space Sparsity: High dimensionality in combined text and graph embeddings posed challenges for simpler models.
